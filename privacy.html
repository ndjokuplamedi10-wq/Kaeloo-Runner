Politique de Confidentialité de Kaeloo Runner

Dernière mise à jour : 14 avril 2026

1. Collecte des données

L'application Kaeloo Runner ne collecte pas directement de données personnelles identifiables (comme votre nom ou votre adresse e-mail). Cependant, nous utilisons des services tiers qui peuvent collecter des informations utilisées pour vous identifier.

2. Services tiers

L'application utilise les services de tiers suivants qui peuvent collecter des données conformément à leurs propres politiques de confidentialité :

  • Google Play Services : Utilisé pour le fonctionnement technique de l'application sur Android.
  • AdMob : Utilisé pour diffuser des publicités. AdMob peut collecter des identifiants publicitaires (IDFA/AAID) pour personnaliser les annonces.
  • Unity : Le moteur de jeu peut collecter des données techniques sur l'appareil pour assurer la stabilité et les performances.

3. Sauvegarde locale

L'application enregistre votre progression (score, argent, personnages débloqués) localement sur votre appareil via les PlayerPrefs. Ces données restent sur votre téléphone.

4. Droits des utilisateurs

L'application propose une fonction "Reset Total" dans les paramètres qui permet de supprimer instantanément toutes les données de progression stockées sur l'appareil.

5. Contact

Si vous avez des questions concernant cette politique, vous pouvez nous contacter via notre page GitHub.
